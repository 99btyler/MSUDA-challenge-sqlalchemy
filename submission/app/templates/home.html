• /api/precipitation - returns the last 12 months of precipitation data
  • /api/stations - returns the stations
  • /api/tobs - returns the last 12 months of temperature data for the most-active station
  • /api/range/<start> - returns the minimum, average, and maximum temperature from the specified range
  • /api/range/<start>/<end> - returns the minimum, average, and maximum temperature from the specified range
